# OpenWrt Snapshot builds for EN7528 Devices

This repository contains the github actions to build and release
OpenWrt for EN7528 devices including:

* DASAN H660GM-A
* Generic initramfs image

**WARNING:** This OS is *minimal*, there is no support for USB yet.
You can communicate with it using UART, Ethernet or WiFi.
